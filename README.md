# App Hub

A centralized launcher for all my personal web apps, providing quick access to each project in one place. Built with **ReactJS** and styled using **shadcn/ui** components.

## Features

- Browse all personal web apps from a single page
- App cards with icons, descriptions, tags, and screenshots
- Animated UI with Framer Motion
- Aurora background effect
- Responsive design with dark mode support
- Light, dark, and system theme toggle

## Getting Started

1. **Clone the repository:**
   ```
   git clone https://github.com/yjaphzs/app-hub.git
   cd app-hub
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the development server:**
   ```
   npm run dev
   ```

4. **Build for production:**
   ```
   npm run build
   ```

5. **Preview the production build:**
   ```
   npm run preview
   ```
